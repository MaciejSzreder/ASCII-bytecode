ASCII bytecode - prototype

The main idea is to use ASCII code points as bytecode.

Interface

No interface. Open console in your browser (probably Ctrl+Shift+I).

Use Machine.execute("code",steps) static method to execute steps steps for code and receive the output as Array.

Create new Machine and use Machine.load("code") method to load code in to memory, Machine.output(()=>action) to catch output (default is console.log), at least call Machine.run(steps) to execute steps steps.
